# 记忆之忆 Mnemosyne

> 一个为 SillyTavern 设计的、对抗记忆污染的事实卡记忆系统  
> **_Mnemosyne_** — 希腊神话中的记忆女神,九位缪斯的母亲

## ✨ 核心特色

记忆插件常见的两大痛点:

- 散文总结派让 LLM 把对话压缩成文字 → **主语漂移**: a 做的事变 b 做的
- 表格记忆派把记忆塞进结构化表格 → **token爆炸 + 操作繁琐**

**记忆之忆**走第三条路 — **事实卡(Fact Card)**:

- 🎯 **结构化主语强制锚定** — 从根源杜绝主语污染
- 🔒 **锁定机制** — 用户确认过的事实,LLM 永远不能擅自修改
- ⚖️ **冲突仲裁面板** — 检测到 a/b 矛盾时弹出对比,用户裁决
- 📍 **来源追溯** — 每张卡片记录来自第几轮哪条消息,永远可查
- 📸 **自动快照** — 污染了一键回滚,数据永远安全
- 📦 **智能归档** — 已完成的承诺、老旧的事件自动归档
- 🌐 **世界书联动** — 一键导出为世界书条目
- 📱 **移动端优先** — 手机 Chrome 完美使用

## 📦 卡片类型

MVP 版本支持 6 种卡片类型,覆盖大部分剧情场景:

| 类型 | 用途 | 更新方式 |
|------|------|---------|
| 👤 身份 | 角色身份/职业/外貌 | 覆盖 |
| ❤️ 关系 | 角色之间的关系 | 覆盖 |
| ⚡ 事件 | 已发生的关键剧情 | 累积 |
| 🤝 承诺 | 谁答应谁什么 | 累积+状态 |
| 💰 债务 | 谁欠谁什么 | 累积+状态 |
| 📦 物品 | 谁持有什么 | 覆盖 |

## 🛠️ 安装

### 方式一:通过 GitHub URL 安装(推荐)

1. 在 SillyTavern 中点击右上角 🧩 扩展图标
2. 找到 **Install Extension** 按钮
3. 粘贴本仓库的 URL:`https://github.com/your-username/st-mnemosyne`
4. 点击安装,刷新页面

### 方式二:手动安装

1. 下载本仓库的全部文件
2. 放到 `SillyTavern/public/scripts/extensions/third-party/st-mnemosyne/` 目录下
3. 重启 SillyTavern

## 🎬 使用流程

### 自动模式(推荐)

1. 安装后打开任意角色卡,正常聊天
2. 每聊 5 轮(可配置),插件会自动调用主对话模型抽取事实卡
3. 每次主对话生成时,自动注入相关记忆到 prompt
4. 如果检测到矛盾,主面板会弹出 ⚠️ 提示,点进去人工裁决

### 手动模式

- **立即抽取**: 主面板点 ⚡ 按钮强制抽取一次
- **手动添加**: 点 + 按钮直接创建一张卡片
- **手动编辑**: 全屏面板里点任意卡片可以修改

### 锁定关键事实

对于不希望 LLM 改动的事实(比如世界观设定、固定关系):

1. 在卡片管理面板找到该卡
2. 点 🔒 锁定按钮
3. 之后无论 LLM 抽取出什么,锁定的卡都不会被自动修改

## ⚙️ 推荐设置

| 设置项 | 推荐值 | 说明 |
|--------|--------|------|
| 抽取间隔 | 5 | 每5轮抽取一次,平衡频率和成本 |
| 抽取风格 | 温和版 | 中文prompt,不易触发审查 |
| 注入位置 | 聊天历史之前 | 作为"已知事实"前置 |
| 最大注入卡片 | 20 | 避免占用过多token |
| 自动快照间隔 | 10 | 每10轮自动备份 |
| 事件归档阈值 | 50 | 超过50轮的事件归档 |

## 🐛 调试

如果遇到问题:

1. 打开浏览器 Console
2. 在「设置 → 调试模式」中勾选 🐛 调试
3. 重新触发问题,Console 会打印详细日志
4. window.Mnemosyne 暴露了调试接口,可以手动调用各个函数
